def rrPython::getCCode ( )

Retrieve a handle for the C code structure, RRCCode.

When cRoadRunner is run in C generation mode its uses a temporary folder to store the generate C source code. This method can be used to obtain the header and main source code after a model has been loaded by using the helper routines (getCCodeSource and getCCodeHeader())

Returns:
Returns null if it fails, otherwise it returns a pointer to the RRCode structure
