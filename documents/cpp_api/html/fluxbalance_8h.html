Detailed Description

basic functions for steady state flux balance analysis of SBML files

This file states flux balance analysis of SBML models (or Stoichiometry matrices) along with flux constraints as linear programming question, which is then solved using lpsolve (http://sf.net/projects/lpsolve).
Author:
Frank T. Bergmann (fbergman@u.washington.edu)
