deque< Rule > rr::NOMSupport::ReorderAssignmentRules ( deque< Rule > &  assignmentRules)

Reorders assignment rules. In SBML assignment rules does not have to appear in the correct order. That is you could have an assignment rule A = B, and a rule B = C. Now the result would differ, if the first rule is evaluated before the second. Thus the rules will be reordered such that this will be taken care of.
