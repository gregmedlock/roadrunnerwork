00019 //   \par
00020 //  LibStructural represents the main class for all structural analysis on
00021 //   either http://sbml.org/ models or directly on a provided stoichiometry matrix.
00022 
00023 //   \par
00024 //  The model can be either analyzed employing QR factorization with householder reflections,
00025 //  or with LU(P) factorization. The QR factorization is the superior method.
00026 //
00027 //   \par
00028 //  For further information please also see:
00029 //\par
00030 //  Vallabhajosyula RR, Chickarmane V, Sauro HM.
00031 //  Conservation analysis of large biochemical networks
00032 //  Bioinformatics, 2005 Nov 29
00033 //  http://bioinformatics.oxfordjournals.org/cgi/content/abstract/bti800v1
00034 
00035 //   \par
